# 🌌 Горизонт Событий — Telegram Бот Оракула

Telegram бот с 64 картами оракула. Пользователь вытягивает случайную карту и получает её название, описание и толкование. История карт сохраняется для каждого пользователя.

---

## 🚀 Установка и запуск

### 1. Клонируй репозиторий
```bash
git clone https://github.com/tajuly/horizon-bot.git
cd horizon-bot
```

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Получи токен бота
- Открой Telegram и найди @BotFather
- Напиши `/newbot` и следуй инструкциям
- Скопируй полученный токен

### 4. Вставь токен в bot.py
Открой `bot.py` и замени строку:
```python
BOT_TOKEN = "ВАШ_ТОКЕН_СЮДА"
```
на свой токен.

### 5. Запусти бота
```bash
python bot.py
```

---

## 🃏 Функции бота

| Команда | Действие |
|---------|----------|
| `/start` | Главное меню |
| `/draw` | Вытянуть случайную карту |
| `/history` | Посмотреть историю карт |
| `/help` | Помощь |

---

## 🖼️ Добавление изображений карт

Чтобы бот показывал изображения карт:

1. Создай папку `cards_images/`
2. Добавь изображения в формате `.jpg`
3. Назови файлы по ID карты: `01.jpg`, `02.jpg`, ..., `64.jpg`

Если изображения нет — бот отправит только текст.

## ☁️ Деплой на сервер (опционально)

Можно запустить бота на сервере 24/7. Например, на Railway, Render или VPS:

```bash
# На сервере
git clone <repo>
cd horizon-bot
pip install -r requirements.txt
python bot.py
```

## 💜 Карты оракула

В колоде 64 карты, каждая с уникальным:
- **Названием** — образное имя карты
- **Описанием** — короткий поэтический образ
- **Толкованием** — практический смысл для жизни

Карты охватывают темы: трансформация, выбор, отдых, смелость, любовь, путь, мудрость и многое другое.
